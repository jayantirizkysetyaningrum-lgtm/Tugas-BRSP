# Capstone GSE25724: T2DM Pancreatic Islets Transcriptomics

## 🎯 Project Overview
**Identifikasi biomarker T2DM melalui analisis microarray pancreatic islets** menggunakan dataset **GSE25724** (7 kontrol + 6 T2DM, Affymetrix HG-U133A).

## 📊 Key Results
| Analysis | Results |
|----------|---------|
| **DEGs** | **4,553 genes** (|logFC|>1, padj<0.05) |
| **Top biomarker** | **CHI3L1** (logFC=2.51, padj=1.2e-10) |
| **GO BP** | **inflammatory response** (p=2.1e-12) |

## 🧬 Biological Findings
- **CHI3L1↑**: Chronic inflammation biomarker
- **Clear clustering**: T2DM vs Control separation
- **Indonesia relevance**: T2DM prevalence 10.8%
